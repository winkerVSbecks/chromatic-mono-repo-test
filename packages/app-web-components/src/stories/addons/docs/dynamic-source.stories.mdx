# Dynamic Source

Stories can use Dynamic Source to display the result of changes to controls.

<Canvas withSource="open">
  <Story
    name="Button"
    component="sb-button"
    args={{
      size: 'large',
      label: 'Button',
      primary: false,
      backgroundColor: undefined,
    }}
  >
    {(args) => html`
      <sb-button
        ?primary="${args.primary}"
        .size="${args.size}"
        .label="${args.label}"
        .backgroundColor="${args.backgroundColor}"
      >
      </sb-button>
    `}
  </Story>
</Canvas>
